# Data Structures & Algorithms

## Big-O Notation

- O(1) -> Constant Time
- O(n) -> Linear Time
- O(n^2) -> Ouadratic Time
- O(n!) -> Factorial Time

### What can cause time in a function

- Operations (+, -, \*, /)
- Comparisions (<, >, ===)
- Looping (for, while)
- Outside Functions call (function())

### Rule Book

1. Worst Case
2. Remove Constants
3. Different terms for inputs
4. Drop Non Dominants
