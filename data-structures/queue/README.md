# Data Structures - Queue

> FIFO - First In First Out

| Method  | Time Complexity |
| :------ | --------------: |
| lookup  |            O(n) |
| enqueue |            O(1) |
| dequeue |            O(1) |
| peek    |            O(1) |

| Pros            | Cons        |
| :-------------- | :---------- |
| Fast Operations | Slow Lookup |
| Fast Peek       |             |
| Ordered         |             |
